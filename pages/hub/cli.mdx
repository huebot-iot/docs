# Hub CLI

## Installation

**Node 16+ is required before installing the CLI**

``` {3} copy
npm install -g @huebot/cli
```

## Commands

* `version` CLI version
* `install` installs the Huebot environment
* `uninstall` uninstalls the Huebot environment
* `upgrade` update Huebot to latest version
* `huebot-version` installed environment version
* `create-admin` create admin user (required for API requests)
* `mqtt-credentials` show MQTT broker credentials
